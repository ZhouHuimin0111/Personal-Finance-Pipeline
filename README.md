# Personal Finance Data Pipeline (个人财务数据流水线)

基于 Python 开发的个人账单自动化处理与分析工具。支持一键提取、清洗并合并支付宝与微信支付导出的账单明细，最终输出标准化的数据透视表和可视化分析图表。

## 核心功能 (Features)

- **双引擎解析**: 自动识别 `.csv` (纯文本) 与 `.xlsx` (Excel 二进制) 格式的账单。
- **智能去噪**: 自动跳过微信/支付宝官方账单头部的非结构化统计说明文字。
- **自动清洗**: 过滤“已退款”、“已关闭”以及“内部转账”等干扰数据，还原真实消费流水。
- **动态可视化**: 结合 `matplotlib` 一键生成月度支出结构饼图与柱状图。
- **隐私隔离**: 采用 `.gitignore` 严格屏蔽真实交易数据上传，保障个人隐私安全。

## 月度记账标准操作流程 (SOP)

为保证财务数据的连贯性，建议每月初执行以下流程：

1. **数据导出 (Extraction)**: 从支付宝/微信导出上月账单，命名为 `alipay.csv` 和 `wechat.xlsx`，放入 `data/` 目录。
2. **自动清洗 (Transformation)**: 运行 `python src/process.py`，生成当月待分类表格。
3. **人工标注 (Labeling)**: 打开生成的表格，参考下方的分类标准完成分类。
4. **汇总入库 (Loading)**: 将分类后的数据追加至年度总账本 `Master_Ledger.xlsx`。
5. **可视化复盘 (Analysis)**: 运行 `python src/analyze.py` 生成支出结构图表。

## 标准化收支分类清单

**支出类核心分类：**
- **餐饮美食**: 一日三餐、零食饮品、交际聚餐
- **交通出行**: 公共交通、长途差旅、打车代步
- **居住日用**: 生活杂项、水电话费、医疗保健
- **学习进阶**: 考试培训、书籍教材
- **数码装备**: 硬件外设、数字服务
- **休闲美容**: 服饰鞋包、娱乐消遣、美妆个护
- **人情社交**: 孝敬长辈、送礼请客
- **特殊偶发**: 突发大支出 (如就医、设备更换)

**收入类核心分类：**
- **生活费用**: 父母给付
- **自我增值**: 奖学金/补助、兼职所得
- **其他收入**: 偶然所得、资产回笼
